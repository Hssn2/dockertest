dockertest Agent + Catalog - Windows Sunucu Kurulum Paketi
==========================================================

Bu paket repoya ihtiyac duymaz. Sadece sunucuda calistir.

KURULUM
-------
1. Bu klasoru sunucuya kopyala (zip'ten cikar)
2. Kur.bat dosyasina sag tik -> Yonetici olarak calistir
3. Docker ilk kez kurulduysa bilgisayari yeniden baslat, Docker Desktop ac
4. Kur.bat'i tekrar calistir (Docker zaten varsa devam eder)

KULLANIM
--------
1. Tarayicida ac: http://SUNUCU-IP:8080
2. Catalog acik: http://SUNUCU-IP:8090
3. Script, catalogdaki son surumu otomatik deploy etmeyi dener
4. Uygulama port 80'de acilir: http://SUNUCU-IP

AYARLAR
-------
config.json dosyasindan degistirebilirsin:
  - AgentImage      : agent docker image (or. ghcr.io/hssn2/dockertest-agent:3)
  - CatalogImage    : catalog docker image
  - PostgresPassword: veritabani sifresi
  - AgentPort       : agent arayuzu (varsayilan 8080)
  - CatalogPort     : catalog arayuzu (varsayilan 8090)
  - AutoDeployLatestFromCatalog: true ise catalogdan son surumu deploy eder

NOT
---
- PostgreSQL, Catalog ve Agent bu kurulumda yuklenir
- Ana uygulama (dockertest) agent uzerinden deploy edilir
- Agent, deploy sirasinda DB ayarlarini otomatik gecirir
- Catalogda surum yoksa otomatik deploy atlanir; cataloga .tar.gz yukleyip agentdan deploy edebilirsin
